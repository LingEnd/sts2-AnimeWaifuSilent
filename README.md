# Anime Waifu Silent

[简体中文](README_zh.md)

## 👩🏾‍🦳 About This MOD
This mod replaces selected **Slay the Spire 2** card portraits for the **Silent** and related pools with anime-style artwork.

The artworks started as character illustrations generated with **Stable Diffusion** on simple-color backgrounds, then were manually edited, repainted, and enhanced with effects in **Photoshop** by [Xpp](https://space.bilibili.com/38777496). This is currently an early preview release, and more portraits will continue to be updated over time.

In addition to portrait replacement, the mod also applies a lightweight re-rendering / smoothing pass to the replacement textures. This helps reduce jagged edges when cards are scaled, hovered, or opened in the large preview view.

It currently includes replacements for:
- 40 Silent cards
- 2 Colorless cards
- 2 Event cards
- 1 Curse card
- 1 Necrobinder card
- 2 Status cards
- 1 Token card

## 📍 Important Notes
1. This mod is an early preview build. If you encounter any issues, please open an issue or contact [@绫末LingEnd](https://space.bilibili.com/187863463).
2. For mod save-related issues, please check the relevant guide or install the [save unification mod](https://www.nexusmods.com/slaythespire2/mods/6).
3. This mod does not affect online gameplay. It is a local mod, so your online teammates do not need to install it.
