# Anime Waifu Silent

[English](README.md)

## 👩🏾‍🦳 关于本 MOD
本 Mod 为 **《杀戮尖塔 2》** 中的 **静默猎手** 及相关卡池替换了二次元风格卡图。

本 Mod 的卡图最初由 **Stable Diffusion** 生成纯色背景立绘，再由 [Xpp](https://space.bilibili.com/38777496) 使用 **Photoshop** 进行修改、补绘与特效制作。目前版本属于早期预览版，后续会继续更新与补完；部分较早制作的卡图未来也可能重制。

除卡图替换外，模组还会对替换后的贴图进行额外的平滑重渲染处理，在卡牌缩放、悬停和大图预览时尽量减轻边缘锯齿，让显示效果更顺滑。

当前版本共替换：
- 40 张静默猎手卡牌
- 2 张无色牌
- 2 张事件牌
- 1 张诅咒牌
- 1 张亡灵契约师卡牌
- 2 张状态牌
- 1 张衍生物卡牌

## 📍 注意事项
1. 本 Mod 为早期预览版本，如遇问题请提交 issue 或联系 [@绫末LingEnd](https://space.bilibili.com/187863463)。
2. 模组存档相关问题请查看相关教程，或安装 [存档统一模组](https://www.nexusmods.com/slaythespire2/mods/6)。
3. 本 Mod 不影响联机玩法，为本地 Mod ，不需要联机队友也安装。
